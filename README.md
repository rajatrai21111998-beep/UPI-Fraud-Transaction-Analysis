# UPI Transaction Fraud Analysis

This project analyzes digital payment transactions to identify fraud patterns and high-risk behaviors.

## Objectives
- Analyze transaction data
- Detect fraud patterns
- Identify high-risk transactions
- Provide insights for fraud monitoring

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- High merchant risk score transactions are more likely to be fraudulent
- Increased transaction frequency indicates suspicious behavior
- Certain payment channels have higher fraud rates
- Abnormal transaction amount deviations indicate potential fraud
